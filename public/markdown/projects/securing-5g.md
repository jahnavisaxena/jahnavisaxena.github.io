# Securing Edge-Cloud Computing in 5G

**Research & Publication** | *International Conference on Advances in Computing, Devices and Feedback (ICACDF 2025)*

This research proposes a multi-layered, zero-trust security framework designed specifically to protect edge-cloud computing configurations in 5G network infrastructures.

## Key Contributions

* **Zero-Trust Network Architecture:** Designed a defensive model that mitigates distributed denial-of-service (DDoS) vectors and prevents data interception at edge gateways.
* **ML Anomaly Detection:** Formulated an artificial intelligence/machine learning anomaly detection subsystem that monitors traffic patterns and edge queries in real-time.
* **Performance Enhancements:** Improved malicious threat detection efficiency by **30%** while minimizing network transmission latency.
* **Framework Design:** Modeled secure authentication schemes and continuous trust evaluation for virtualized network functions (VNFs) at edge datacenters.
